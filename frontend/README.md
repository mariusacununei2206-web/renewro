# Frontend RenewRO (React)

Aici va sta aplicatia React (MVP, fara login):
- simulare interactiva (alegi judet + putere instalata)
- harta choropleth a regiunii Moldova
- grafice de productie si indicatori economici (NPV, IRR, LCOE, payback)
- slider de autoconsum + toggle-uri de senzitivitate

Comunica cu backend-ul (PyCharm) prin HTTP (API).
Se deschide in VS Code / Visual Studio.
